Box Order Management System

Simplifying the box ordering process with real-time inventory checks and efficient quotations.

Features About Go to App

Key Features

Real-Time Quotation

Instantly generate quotations based on box size, type, and quantity.

Inventory Check

Automatically checks available stock during the quotation process to avoid order cancellation or delays.

Order Management

Easy order tracking, payment processing, and invoice generation for both clients and staff.

About This Project

This project was inspired by a personal experience of dealing with long queues and inventory issues while purchasing boxes. The goal is to streamline the box ordering process by providing real-time quotations and inventory checks, minimizing delays, and enhancing customer satisfaction.

This is a Portfolio Project for Holberton School.
